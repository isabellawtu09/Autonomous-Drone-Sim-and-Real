# Autonomous-Drone-Sim-and-Real 

## ```Sim``` folder contains current progress made with ardupilot iris_gimbal model

**Requirements** :


Ardupilot SITL (Software in the loop) and Ardupilot_gazebo plugin for built-in models 
https://medium.com/@sanjana_dev9/how-to-set-up-ardupilot-sitl-with-gazebo-for-drone-simulation-a0d15e19b8e3
 
(our sim requires that these models be tweaked in some form to model the real drone's equipment

##  ```Hardware_real``` folder contains the mirrored progress done in sim to the actual drone 

Navigate to both of these folders and create your own branch when creating new changes to either folder.


##  ```Text_Interface``` folder contains the boiler template code for our current interface to work with both sim and real hardware
